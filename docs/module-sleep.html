    sleep

    (require("sleep"))(ms) → {Promise}

    Pause a current function's execution without stopping up the entire event loop. Similiar to a setTimeout, but more inline.

    Parameters:
    Name Type Description
    ms number

    amount of time in milliseconds that the function should pause execution.
    Source:
    • sleep.js, line 15
    Returns:

    awaited until the timer runs out.

    Type
    Promise
    Example 
    console.log('First log');

sleep(2000);

console.log('second log 2000 milliseconds later);
