    nestedLoop

    (require("nestedLoop"))(collection1, collection2, callback) → {void}

    Shortcut to writing nested loops. Works with both arrays and objects.

    Parameters:
    Name Type Description
    collection1 Array | Object

    Array or Object to loop over
    collection2 Array | Object

    Array or Object, this collection is looped over in its entirety every iteration of collection1
    callback function

    callback function to be called per iteration of collection2, this function is ran n(collection1's length * collection2's length) times. Both collection indexes that are the current iteration are passed to this function.
    Source:
    • nestedLoop.js, line 19
    Returns:

    None

    Type
    void
    Example 
    let data1 = [1, 2, 3, 4];
let data2 = [5, 6, 7, 8];

nestedLoop(data1, data2, function(i, j) {
 console.log(i + j);
});
